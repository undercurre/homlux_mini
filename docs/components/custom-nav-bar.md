# 通用导航栏

## 简介

通用导航栏包含了一个插槽和默认的 title，如果 title 不满足可以通过插槽自定义

## 属性

| 参数 name | 类型   | 必传  | 默认值 | 描述           |
| --------- | ------ | ----- | ------ | -------------- |
| title     | string | false | -      | 默认样式的标题 |

## 插槽

| 插槽 name | 描述                                          |
| --------- | --------------------------------------------- |
| -         | 如果 title 默认样式不符合可以通过该插槽自定义 |
